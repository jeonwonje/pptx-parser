# pptx-parser
Quick .pptx parser that uses [JSZip](https://github.com/Stuk/jszip) and [node-xml2js](https://github.com/Leonidas-from-XIV/node-xml2js).

## v2
- Support for browser
- 3 outputs, title, text and images
